## Grammaire

### Like it/that

* plaire = to like; similar to gustar
* ça + me/te/lui/nous/vous/leur + plaît (singular) or plaisent (plural)
  * Example: Il me plaît. (I like it)

### suivre (to follow, take [a course])

|                     |                  |
| ------------------- | ---------------- |
| je **suis**         | nous **suivons** |
| tu **suis**         | vous **suivez**  |
| il/elle/on **suit** | ils **suivent**  |

Past participle: **suivi**

### Pronominal verbs with indirect objects en passé composé

Unlike when the subject represents the direct object (e.g. Ils se sont embrassés), with indirect objects, we don't match the verb to agree.

* Example: Ils se sont parlé au téléphone.

### Exceptions with vous

* vous always ends in -ez except for "vous êtes", "vous faites", and "vous dites"

### lire, dire, écrire

#### lire (to read)

| ------------------ | -------------------- |
| je **lis**         | nous **lisons**      |
| tu **lis**         | vous **lisez**       |
| il/elle/on **lit** | ils/elles **lisent** |

Irregular past participle: **lu**

#### dire (to say, to tell)

|                    |                      |
| ------------------ | -------------------- |
| je **dis**         | nous **disons**      |
| tu **dis**         | vous **dites**       |
| il/elle/on **dit** | ils/elles **disent** |

Irregular past participle: **dit**

#### écrire (to write)

|                      |                        |
| -------------------- | ---------------------- |
| j'**écris**          | nous **écrivons**      |
| tu **écris**         | vous **écrivez**       |
| il/elle/on **écrit** | ils/elles **écrivent** |

Irregular past participle: **écrit**

#### Indirect object pronouns with lire, dire, and écrire

* lire, dire, and écrire are often used together because the subject communicates to someone else.
  * example: Le prof nous lit des poèmes. (The professor reads poems to us.)
