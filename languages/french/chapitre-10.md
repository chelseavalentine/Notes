## Grammaire

### Gift-giving verbs: offrir and ouvrir

* **offrir** (to offer; to give [as a gift]) and **ouvrir** (to open) are conjugated like -er verbs in the past tense
* they have irregular past participles: "j'ai offert" and "ils ont ouvert"

### Objects in gift-giving

J'offre une belle jupe à ma mere.

* the **direct object** is the item being offered to someone (une belle jupe)
* the **indirect object** is the person who receives the item (ma mère)

* replace the indirect object with a pronoun lui (one person) or leur (multiple people)
  * Je leur offre des fleurs.

### croire and recevoir: subordinate clauses

Irregular imparfait stems of voir, croire, and recevoir: voy-, croy-, recev-

Passé composé: vu, cru, reçu

#### croire (to believe)

|                      |                       |
| -------------------- | --------------------- |
| je **crois**         | nous cro**y**ons      |
| tu **crois**         | vous cro**y**ez       |
| il/elle/on **croit** | ils/elles **croient** |

* croire en...
  * bonté (goodness)
  * abilité
  * dieu (god)
  * etc.
* croire à… ('y' can replace à + [nom] for what the person believes in, in the responses)
  * quelque chose
  * quelqu'un
  * [nom]
* croire que...
  * quelque chose
  * quelqu'un
  * [phrase]

#### recevoir (to receive)

|                       |                         |
| --------------------- | ----------------------- |
| je re**ç**ois         | nous recevons           |
| tu re**c**ois         | vous recevez            |
| il/elle/on re**ç**oit | ils/elles re**ç**oivent |

### Relative clauses with qui, que, and où

* qui is used when referring to the subject, and que (qu') is used when referring to the object
  * C'est un personnage que les enfants adorent.
  * Voici un cadeau qui vient de Paris.
* use où with a location or time
  * Décembre est un mois où il y a beaucoup de fêtes.
* when using que (only) in the passé composé, the verb agrees with the object in gender and plurality
  * Ce sont les cadeaux (m. pl.) que j'ai achetés.
  * C'est la dinde que j'ai achetée.

### Imperative for giving commands

* imperative forms are the same as the present tense, except in -er verbs since you drop the -s for the tu form
  * Va chercher dans ta chambre!
  * Finis tes devoirs!
* use nous to make suggestions: it's the same as saying "Let's..."

### Commands to the general public

* used in recipes, signs, etc: use the infinitif and note that "ne pas" precedes the infinitif in negation
* add "Veuillez…" & "Prière de…" for please, and "Défense de…" & "Interdit/Interdiction de…" for prohibition
  * Pour paiement par chèque, veuillez présenter un pièce d'identité.
  * Ne pas toucher.

### être, avoir, and savoir in the imperative

| Verb   | tu    | vous   | nous    |
| ------ | ----- | ------ | ------- |
| être   | sois  | soyez  | soyons  |
| avoir  | aie   | ayez   | ayons   |
| savoir | sache | sachez | sachons |

### Uses and forms of adverbs (made from adjectives)

* 5 types of adverbs: adverbs de...

  * degré: énormément, beaucoup, tellement (so much), assez, trop
  * fréquence: toujours, souvent, quelquefois, rarement, parfois
  * temps: hier, aujourd'hui, maintenant, bientôt, demain
  * lieu: ici, là, là-bas, ailleurs (elsewhere), partout (everywhere)
  * manière: bien, mal, soigneusement

* form adverb by using the feminine form if the masculine form ends in a consonant

  | voyelle (masculine)  | consonne (feminine)                 |
  | -------------------- | ----------------------------------- |
  | joli -> joliment     | distrait/distraite -> distraitement |
  | absolu -> absolument | soigneux/soigneuse -> soigneusement |

* exceptions:

  * -e -> -ément
  * -ant -> -amment
  * -ent -> -emment
